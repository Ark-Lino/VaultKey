What is VaultKey?

VaultKey is a Firefox extension that generates two-factor authentication (2FA/TOTP) codes entirely offline. No data ever leaves your browser. No accounts, no cloud, no tracking. Your secrets stay on your machine, period.

Import your existing accounts from Google Authenticator via migration links, or add them manually. VaultKey handles the rest — generating fresh codes every 30 seconds with military-grade HMAC-SHA1 cryptography.

How it's made

Web Crypto API

Native browser cryptography for HMAC-SHA1 TOTP generation. No external crypto libraries.

Manifest V3

Latest Firefox extension format with strict CSP. No remote code execution possible.

Protobuf Decoder

Custom binary parser for Google Authenticator migration links. No dependencies.

QR Code ISO/IEC 18004

Battle-tested QR generation for exporting accounts to authenticator apps.
